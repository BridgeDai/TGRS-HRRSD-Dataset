# Construction

The COCO format data set construction is as follows:

```
 |Anns
 |------|train.json
 |------|val.json
 |------|test.json
 |------|tiny.json
 |Imgs
 |------|1.jpg
 |------|2.jpg
 |------|...
```

In this part, we provide the COCO formatted-Annotations of the HRRSD data set.

The annotations are under the Anns folder.
